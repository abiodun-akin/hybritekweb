# 05 Due Diligence Data Room Checklist

Use this checklist to assemble all supporting evidence before final submission and follow-up diligence.

## A. Company And Legal
- [ ] Company registration documents
- [ ] Founder IDs and role declarations
- [ ] Cap table (current and post-option, if any)
- [ ] Key contracts or agreements summary
- [ ] IP ownership statement

## B. Team
- [ ] Founder bios (1 page each)
- [ ] Team org chart (current + next 12 months)
- [ ] Skills matrix (product, ops, growth, partnerships)
- [ ] Advisory network list

## C. Product
- [ ] Product demo video (3-5 minutes)
- [ ] Live product links (frontend/backend where appropriate)
- [ ] Feature map and roadmap
- [ ] Security/privacy statement
- [ ] Architecture one-pager (can reference existing project docs)

## D. Traction And Impact Evidence
- [ ] User numbers (registered, active, retained)
- [ ] Funnel metrics (signup to active workflow)
- [ ] Pilot outcomes and case examples
- [ ] User testimonials (farmer/vendor)
- [ ] Impact baseline + targets from MEL framework

## E. Market And Business Model
- [ ] Problem sizing note (TAM/SAM/SOM, if available)
- [ ] Segment definitions and value propositions
- [ ] Revenue model assumptions
- [ ] Unit economics snapshot (if early, include assumptions)
- [ ] Competitive positioning table

## F. Financial
- [ ] 12-month operating plan
- [ ] Budget use plan tied to milestones
- [ ] Financial controls summary
- [ ] Funding history and current runway

## G. Programme-Specific Readiness
- [ ] Eligibility mapping completed (`01_ELIGIBILITY_AND_FIT.md`)
- [ ] Narrative finalized (`02_PROGRAMME_PITCH_NARRATIVE.md`)
- [ ] MEL and KPIs finalized (`03_IMPACT_AND_MEL_FRAMEWORK.md`)
- [ ] Implementation plan finalized (`04_IMPLEMENTATION_PLAN_AND_BUDGET.md`)
- [ ] Application form responses finalized (`06_APPLICATION_FORM_DRAFT_RESPONSES.md`)

## H. Submission QA
- [ ] All claims have evidence or are clearly labeled as targets
- [ ] Tone is concise, confident, and impact-focused
- [ ] Metrics are consistent across all documents
- [ ] Deadline and submission link verified
- [ ] Final PDF export pack prepared (if needed)
